## 📱 Overview

Tailor Desk is a powerful mobile application designed specifically for tailors to manage their business efficiently. From customer management to order tracking and financial reporting, it provides everything a tailor needs to run their shop digitally.

### 🎯 Key Features

| Feature | Description |
|---------|-------------|
| 👤 **Customer Management** | Add, edit, search, and delete customers |
| 📏 **Measurement Tracking** | Record and update customer measurements (chest, waist, hip, etc.) |
| 📦 **Order Management** | Track pending and completed orders |
| 💰 **Billing System** | Generate bills and manage order history |
| 📊 **Income Reports** | Track revenue, average orders, and monthly breakdowns |
| 👨‍💼 **Profile Management** | Update tailor profile and shop details |
| 🔐 **Authentication** | Secure login and signup system |
